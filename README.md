# TwitterGeoGraph
twitterを用いた位置情報Webアプリ 

# 概要
① 
twitterAPIを用いてユーザのツイートからユーザの位置を取得してwebアプリに(最終的にはスマホアプリ)の地図状にtwitterアカウントのアイコンマークを表示させるシステム 

② 
位置情報とその発言時のネガポジから、その場所(店)の良し悪しの情報も見れるようにする 

③ 
学生ごとに分類して生息地分布的なのを作成する
